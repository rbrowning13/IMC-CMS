Impact Medical CMS - Initial Skeleton

How to run:

1. Create and activate a virtual environment, e.g.:

   python3 -m venv .venv
   source .venv/bin/activate   # macOS/Linux
   # or .venv\Scripts\activate on Windows

2. Install dependencies:

   pip install -r requirements.txt

3. Run the app:

   python run.py

4. Open http://127.0.0.1:5000 in your browser.

This is just the starting point (Milestone 0).
Claims and Carriers pages are placeholders that we'll expand with real data and forms.
